# Medisearch Life Sciences  
## Stock Return & Goods Issue Management System

A clean and lightweight browser-based application designed to simplify the management of **stock return** and **goods issue** records for day-to-day business operations.

Built for quick access and ease of use, this tool allows users to create, save, print, search, and export reports directly from the browser — without requiring any external database or server setup.

---

# Features

- Create stock return and goods issue reports
- Add expiry products and issued items dynamically
- Automatic total calculations
- Save reports locally in the browser
- View, search, and filter saved reports
- Print current and previous reports
- Export reports to Excel format
- Dark mode and light mode support
- Fully browser-based and easy to use

---

# Technologies Used

- HTML5
- CSS3
- JavaScript
- Browser Local Storage (`localStorage`)
